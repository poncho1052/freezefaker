## Phase 1: Core Prototype

Goal:

Test whether the core loop is fun.

Scope:

- One graybox map
- NPC walking and freezing
- Faker movement
- Watcher accusation
- Simple win/loss

Success criteria:

- Red Light moment feels good
- Watcher can identify suspicious behavior
- Faker wants to play again
- Viewers can guess who is human

---

## Phase 2: Vertical Slice

Goal:

Make the game feel like Freeze Faker, not just a prototype.

Scope:

- Station Front visual pass
- Disguise actions
- NPC Sync
- Better freeze poses
- Better UI
- Result screen
- Basic playtest loop

Success criteria:

- 30 minutes of play creates multiple funny moments
- Both Faker and Watcher roles are fun
- Short-video clips are possible

---

## Phase 3: Commercial Alpha

Goal:

Build the foundation of a sellable game.

Scope:

- 3 maps in playable state
- Online room creation
- Private rooms
- Custom settings
- Tutorial
- Spectator mode
- 10+ disguise actions
- 15+ NPC behaviors

Success criteria:

- Friend groups can play for 30〜60 minutes
- Streamers can understand and run a session
- Core UI is stable
- Multiplayer is reliable enough for broader testing

---

## Phase 4: Beta

Goal:

Content, balance, and platform readiness.

Scope:

- Steam integration
- Achievements
- Localization EN/JA
- Controller support
- Performance optimization
- QA pass
- Store assets
- Trailer

Success criteria:

- External testers can play without explanation
- Bugs are manageable
- Store page can be prepared
- Trailer footage is available

---

## Phase 5: Launch

Goal:

Release on Steam.

Scope:

- Steam page live
- Trailer released
- Press kit ready
- Discord / social channels ready
- Creator outreach
- Launch build uploaded

Success criteria:

- Clear positioning
- Short-video-ready clips
- Streamer-friendly onboarding
- Positive first reviews
