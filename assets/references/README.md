# Reference Assets

This folder is for visual and design references used during Freeze Faker development.

Suggested structure:

- `logo/` — logo concepts and title treatments
- `key_visual/` — key art and hero image references
- `ui/` — HUD, lobby, result, and UI direction references
- `characters/` — NPC and player character concept references
- `world/` — environment, map, station-front, and city references

Generated concept images should be treated as references, not final production assets.

Do not rely on AI-generated text embedded inside images as final UI text. Recreate UI text manually in production.
